    Ellipse Mathematics

    Deep dive into parametric ellipses with rotation.

    Parametric Ellipse Equations

    Unrotated ellipse centered at origin:

    x(t) = rx  cos(2t)
y(t) = ry  sin(2t)

Where:
  t  [0,1]  : Parameter
  rx         : Horizontal radius (semi-major or semi-minor)
  ry         : Vertical radius
  2t        : Angle in radians (0 to 2)

    With Rotation

    Apply 2D rotation matrix with angle :

    [x']   [cos()  -sin()] [x]
[y'] = [sin()   cos()] [y]

Expanded:
x'(t) = x(t)cos() - y(t)sin() + cx
y'(t) = x(t)sin() + y(t)cos() + cy

Where:
  (cx, cy) : Center coordinates
          : Rotation angle (radians)

    Full Formula

    x'(t) = cx + rxcos(2t)cos() - rysin(2t)sin()
y'(t) = cy + rxcos(2t)sin() + rysin(2t)cos()

    Circle as Special Case

    When rx = ry = r:

    x(t) = cx + rcos(2t)
y(t) = cy + rsin(2t)

    Rotation has no visual effect!

    Eccentricity

    Measure of "ovalness":

    e = sqrt(1 - (b/a))

Where:
  a = max(rx, ry) : Semi-major axis
  b = min(rx, ry) : Semi-minor axis

e = 0     : Perfect circle
0 < e < 1 : Ellipse
e = 1     : Parabola (limit)

    Perimeter Approximation

    No exact formula! Ramanujan's approximation:

    P  [3(a+b) - sqrt((3a+b)(a+3b))]

    Better approximation:

    h = ((a-b)/(a+b))
P  (a+b)(1 + 3h/(10 + sqrt(4-3h)))

    Area

    Exact formula:

    A =   rx  ry

    For circle: A = r

    Angle vs Parameter

    Not the same! For non-circles: - Parameter t uniform speed around ellipse - Angle non-uniform (faster on short axis)

    # Parameter t (uniform)
point = ellipse.point_at(0.25)  # 25% around ellipse

# Direct angle (non-uniform)
point = ellipse.point_at_angle(90)  # At 90

    Focal Points

    For ellipse with a > b:

    Focal distance from center: c = sqrt(a - b)
Foci at: (cx  c, cy)  # Before rotation

Distance sum property:
  distance(P, F) + distance(P, F) = 2a
